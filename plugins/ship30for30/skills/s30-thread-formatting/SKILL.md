# Day 20: Thread Formatting

> **Trigger:** "Build Day 20" or "Thread Formatting"

## What You'll Build

A complete Twitter/X thread (5-9 tweets) with strong hook, 3-5 main points, TL;DR recap, and call-to-action using one of 7 proven viral formats.

---

## Build Flow

1. **Read config** — Load `s30-config.md` for student's niche, topics, and Notion DB ID
2. **Read template** — Load `references/D20-content.md` for 4-part anatomy, 7 hook patterns, and 7 viral formats
3. **Customize** — Help student choose format, develop hook, write main points with clear progression
4. **Deliver to Notion** (if connected):
   - Find Day 20 row in student's Notion database
   - Fill the **Template** column with complete thread (formatted as tweets)
   - Set **Status** to "In Progress"
5. **Fallback** (no Notion): Save to `workspace/day-20-thread.md`
6. **Confirm** — "Your thread is tight. Hook stops the scroll, each tweet lands, TL;DR recaps perfectly."

---

## Review Criteria

When student shares their thread, check:
- **Strong hook tweet** — Uses one of 7 proven patterns; makes reader want to click "Read"
- **Each tweet standalone** — Could screenshot any tweet and it still makes sense
- **Proper thread structure** — Has 4 parts: hook, main points, TL;DR, CTA

Word count: Varies (typically 5-9 tweets, 1,000-1,500 characters total)

Give 1-2 specific notes on hook strength or CTA clarity.

---

## On Completion

Update Notion Status to "Thread Complete" (or note in config). Move to Day 21 (long-form platform posts). Ask if they want to continue.
