# Day 30: Graduation

> **Trigger:** "Build Day 30" or "Graduation Plan"

## What You'll Build

Reflect on 30 days of progress, celebrate your wins, and create an Idea File system (1 Topic × 7 Subtopics × 20 Questions = 140 ideas) for ongoing content production.

---

## Build Flow

1. **Read config** — Load `s30-config.md` for student's 30-day performance metrics
2. **Read template** — Load `references/D30-content.md` for Idea File and Full-Stack Writer frameworks
3. **Reflect** — Review student's top 5 posts, biggest learnings, growth trajectory
4. **Plan** — Draft Idea File structure (1 topic, 7 subtopics, 20 questions per subtopic)
5. **Deliver to Notion** (if connected):
   - Find Day 30 row in student's Notion database
   - Fill the **Template** column with Idea File seed questions
   - Add **Next Steps** (Full-Stack Writer path, email list strategy, monetization plan)
   - Set **Status** to "Shipped"
6. **Fallback** (no Notion): Save to `workspace/day-30-graduation-plan.md`
7. **Confirm** — "Your 30 days are complete. Your Idea File is ready. Here's your Full-Stack Writer roadmap."

---

## Review Criteria

When student completes Day 30, check:
- Clear reflection on biggest learnings from 30-day journey
- Idea File properly structured (1 topic clearly stated, 7 subtopics identified)
- At least 40-50 seed questions generated in Idea File
- Student identifies 1-2 monetization strategies (email list, digital products, services)
- Next 90-day action plan defined (consistency cadence, audience growth, revenue targets)

---

## On Completion

Update Notion Status to "Shipped". This is graduation day—celebrate the student's achievement and send them off with confidence toward their Full-Stack Writer future.
