# Day 11: Twist The Knife

> **Trigger:** "Build Day 11" or "Twist the knife"

## What You'll Build

Write an emotional hook opener that identifies reader pain, twists it (amplifies the feeling), then transitions to a solution.

---

## Build Flow

1. **Read config** — Load `s30-config.md` for student's niche, topics, and Notion DB ID
2. **Read template** — Load `references/D11-content.md` for Day 11 writing template
3. **Customize** — Fill template with student's topic and audience pain points from config
4. **Deliver to Notion** (if connected):
   - Find Day 11 row in student's Notion database
   - Fill the **Template** column with customized template
   - Set **Status** to "In Progress"
5. **Fallback** (no Notion): Save to `workspace/day-11-template.md`
6. **Confirm** — "Your Day 11 template is ready in [Notion/local file]. Write your draft, then tell me when you're done for feedback."

---

## Review Criteria

When student shares their draft, check:
- **Identifies specific reader pain** — Headline/opening names a real problem their audience faces (not abstract)
- **Twists the knife emotionally** — Uses specific, visceral language that amplifies the pain/stakes (words like "crash," "destroy," "sabotage")
- **Transitions to solution** — Clearly pivots from problem to promised solution or insight
- Word count (target: ~250, max 300)
- One clear idea, no fluff

Give 2-3 specific, actionable notes. Don't rewrite for them.

---

## On Completion

Update Notion Status to "Shipped" (or note in config). Ask if they want to move to the next day.
