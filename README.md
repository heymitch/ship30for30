# Ship 30 for 30

Write and publish 30 pieces of content in 30 days. 35 skills covering every aspect of the Ship 30 for 30 writing system.

## Install

Add the marketplace, then install the plugin:

```
/plugin marketplace add heymitch/ship30for30
/plugin install ship30for30@ship30-marketplace
```

One install — all 35 skills appear automatically.

## What's Inside

- **Writing System** — atomic essays, headlines, hooks, frameworks, structures
- **Idea Generation** — endless idea machine, topic mining, audience research
- **Editing & Quality** — self-editing, clarity checks, readability scoring
- **Publishing Strategy** — platform selection, distribution, repurposing
- **Tutor & Builder** — guided learning mode and hands-on writing mode
